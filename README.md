# Simple-news-website

一个简单的node+bootstrap做的小的练习项目，能实现简单的登录注册，session实现。数据库用的MySQL

> node

  主要练习express框架、express-session组件、body-parser组件、ejs模板引擎的使用

> 前端页面

  使用bootstrap框架搭建，美观简洁，适配移动端

> 数据库

  使用的是mysql数据库，练习了node连接mysql数据库和对数据库的简单操作
